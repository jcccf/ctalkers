Joseki is an HTTP engine that supports the SPARQL Protocol and the SPARQL RDF Query language. SPARQL is developed by the W3C RDF Data Access Working Group.

Joseki Features:

  • RDF Data from files and databases
  • HTTP (GET and POST) implementation of the SPARQL protocol
